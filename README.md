# Midterm_EquipmentRental_Team1

Steps to run application:
1. Open a terminal window
2. Navigate to API project Midterm_EquipmentRental_Team1/Midterm_EquipmentRental_Team1_API
3. Execute command 'dotnet run'
4. In a new Terminal window, navigate to UI project Midterm_EquipmentRental_Team1/Midterm_EquipmentRental_Team1_UI
5. Execute command 'dotnet run'
6. Navigate to browser and reach https://localhost:7025
